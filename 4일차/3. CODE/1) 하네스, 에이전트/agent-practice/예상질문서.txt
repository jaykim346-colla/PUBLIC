[화주 문의 샘플 모음 - 에이전트 실습용]
※ 아래 문의들을 하나씩 에이전트에게 입력하며, 에이전트가
   '무엇을 묻는지 스스로 판단해' 알맞은 데이터를 찾아 답하는지 확인하세요.

문의 1) 추적 질문
"안녕하세요, 저희 부킹번호 HMMU1234567 화물이 지금 어디쯤 있나요? 언제 도착하나요?"

문의 2) 운임 질문
"부산에서 LA로 40HC 한 대 보내려는데 선사별 운임이 어떻게 되나요? 제일 싼 곳으로 알려주세요."

문의 3) 스케줄 질문
"부산-싱가포르 구간 제일 빨리 가는 배가 뭔가요?"

문의 4) 복합 질문 (판단 난이도 ↑)
"코리아텍입니다. 우리 화물 상태 확인 좀 부탁드리고, 혹시 마감 임박한 거 있으면 같이 알려주세요."

문의 5) 모호한 질문 (에이전트가 되물어야 정상)
"운임 좀 알려주세요."
   → 구간/컨테이너 정보가 없으므로 에이전트가 추가 정보를 되물으면 정답.
